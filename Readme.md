# 💊 Medication Reminder App

A responsive Medication Reminder web application built with **HTML**, **CSS**, and **JavaScript**. The application helps users organise their medications, receive reminder notifications, and track whether medications have been taken.

This project is currently a **frontend application** that stores data using the browser's **Local Storage**. A full-stack version with user authentication and cloud storage is planned.

---

## ✨ Features

### Medication Management

- Add a new medication
- Edit an existing medication
- Delete a medication
- Mark medication as taken
- Clear all medications

### Reminder Features

- Browser notification reminders
- Reminder sound
- Snooze reminder for 5 minutes
- Repeat reminders:
  - Daily
  - Weekly
  - Every 8 hours

### Organisation

- Search medications
- Medication categories:
  - Morning
  - Afternoon
  - Evening
  - Night

- Upload a medication photo

### Dashboard

View useful medication statistics, including:

- Total medications
- Medications taken
- Missed medications
- Upcoming medications

### Additional Features

- Calendar schedule view
- Dark mode
- Local user profiles
- Export medication schedule
- Print medication schedule
- Responsive design

## 🚀 Getting Started

```bash
git clone https://github.com/Tessydesigns/Medication-reminder-Application.git
```

Open the project folder in **Visual Studio Code**.

Launch `index.html` in your preferred web browser.

No installation or additional packages are required.

---

## 📖 How to Use

1. Enable browser notifications.
2. Enter the medication name.
3. Enter the dosage.
4. Select the reminder time.
5. Choose a medication category.
6. Select the repeat interval.
7. Add instructions (optional).
8. Upload a medication photo (optional).
9. Click **Add Medication**.
10. Use the search bar to find medications.
11. Mark medications as taken after taking them.
12. Export or print your medication schedule if needed.

---

## 💾 Data Storage

This application currently uses the browser's **Local Storage**.

This means:

- Medication data remains available after refreshing the page.
- Data is stored only on the current browser and device.
- Data is not synchronised across different devices.
- Clearing browser storage will remove all saved medications.
